# vagrant Overview

## General Commands

```sh
vagrant init # Create a configuration template file (Vagrantfile)
vagrant up # Run a VM
vagrant reload # reload a VM after make changes
vagrant ssh # ssh access to a VM
vagrant box list # list of VMs 
vagrant box remove # remove a VM
vagrant global-status # list of running VMs
vagrant destroy # run in the VM folder. Result: VM is stopped and deleted

vagrant plugin install vagrant-docker-compose # docker compose installation
```
